Launch

Publish this site with GitHub Pages in a few minutes.

  1. Create a GitHub repository and add these files.
  2. Push the repository to the `main` branch.
  3. Open your repo settings and enable GitHub Pages from the root of `main`.
  4. Visit the Pages URL GitHub gives you and share it.
